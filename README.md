# Trimless LED Website

Bu proje, minimalist ve modern aydınlatma çözümleri sunan **Trimless LED** markası için geliştirilmiş, yüksek performanslı bir landing page projesidir.

## 🚀 Teknolojiler
- **Framework:** [Next.js](https://nextjs.org/)
- **Deployment:** [Cloudflare Pages](https://pages.cloudflare.com/)
- **Styling:** Tailwind CSS
